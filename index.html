Gulp

What is it ?

Gulp is a toolkit for automating painful or time-consuming tasks in your development workflow, so you can stop messing around and build something.
gulp.task('css', function(){
  return gulp.src('client/templates/*.less')
    .pipe(less())
    .pipe(minifyCSS())
    .pipe(gulp.dest('build/css'))
});

Why using Gulp ?

  • Gulp solves the problem of repetition
  • Automate tasks such as :
    • Bundling and Minifying multiple files
    • Refreshing the browser ( hot-reload)
    • Quickly run unit testing
    • Compile code ( Less / Sass )
    • Copying files and cleaning folders
  • Simple to use thanks to the task system
